# Assignment 07 - Titanic Data Visualization

## Overview
This assignment focuses on visualizing the Titanic dataset using Matplotlib and Seaborn. 
It includes bar charts, boxplots, heatmaps, pie charts, histograms, scatterplots, and line plots.

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn

## Goals
- Load and inspect the dataset
- Explore distributions and relationships
- Create multiple types of visualisations
- Draw insights from visual patterns
