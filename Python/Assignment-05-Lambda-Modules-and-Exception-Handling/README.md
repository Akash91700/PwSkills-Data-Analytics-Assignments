# Assignment 05 - Lambda Functions, Modules and Exception Handling

## Overview
This assignment focuses on advanced Python concepts including lambda functions, module usage, and structured exception handling. It strengthens problem-solving skills and code reusability practices.

---

## Concepts Covered

- Lambda functions
- map() and filter()
- math module
- random module
- Custom module creation
- Exception handling (ValueError, ZeroDivisionError)
- User input validation

---

## Difficulty Levels

### Easy Level
- Understanding lambda functions
- Square of a number using lambda
- Using the math module to print the value of pi

### Medium Level
- Doubling numbers using lambda
- Division with exception handling
- Generating random numbers

### Hard Level
- Extracting even numbers using filter()
- Handling multiple exceptions
- Creating and importing a custom module

After completing this assignment, you will be able to:
