# Assignment 08 - Data Analysis Concepts

## Overview
This assignment focuses on fundamental data analysis concepts that every analyst should understand.  
It covers stages of analysis, types of analysis, skills required, and key statistical techniques.

## Topics Covered
- Purpose of Data Analysis
- Key Skills of a Data Analyst
- Types of Analysis (Descriptive, Diagnostic, Predictive, Prescriptive)
- Work Cycle of a Data Analyst
- Regression Analysis
- Factor Analysis
- Quantitative vs Qualitative Data
