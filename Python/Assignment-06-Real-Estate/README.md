# Assignment 06 - Real Estate Data Cleaning and Preparation

## Overview
This assignment focuses on cleaning, transforming, and analyzing a Real Estate dataset using Pandas.  
The objective is to handle missing values, remove duplicates, perform feature engineering, and generate meaningful business insights.

---

## Dataset Information

The dataset contains property-level information including:

- Property_ID
- Location
- Price
- Bedrooms
- Bathrooms
- Area_sqft
- Year_Built

---

## Tools & Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## Key Tasks Performed

### 1. Data Exploration
- Loaded dataset
- Checked shape and column names
- Identified missing values
- Inspected data types

### 2. Data Cleaning
- Filled missing values using median and mode
- Removed duplicate records
- Dropped invalid rows
- Converted data types where necessary

### 3. Feature Engineering
- Created new column: `Price_per_sqft`
- Created new column: `Property_Age`
- Categorized properties as:
  - Expensive
  - Affordable

### 4. Data Analysis
- Location-wise average price
- Most expensive properties
- Size-based analysis
- Sorted data for better insights

### 5. Business Insights
- Identified high-value locations
- Compared property size vs price
- Evaluated modern vs old property trends
