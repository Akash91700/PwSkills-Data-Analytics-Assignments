# SQL Assignment 01 - Introduction and Basic Queries

## Overview

This assignment introduces the fundamentals of SQL including database creation, table creation, filtering data, modifying records, and applying constraints.

SQL (Structured Query Language) is used to interact with relational databases and perform operations such as retrieving, inserting, updating, and deleting data.

---

# Topics Covered

## Basic SQL Operations

- CREATE DATABASE
- CREATE TABLE
- SELECT
- INSERT
- UPDATE
- DELETE

These commands allow developers and analysts to manage and manipulate database data effectively.

---

## Filtering Data

- WHERE clause
- Logical operators (AND, OR)
- Comparison operators
- DISTINCT keyword

These help in retrieving specific data from tables.

---

## Data Manipulation (CRUD Operations)

CRUD operations represent:

- Create → INSERT
- Read → SELECT
- Update → UPDATE
- Delete → DELETE

These operations are the core of database interaction.

---

## SQL Constraints

Constraints enforce rules on database columns to maintain data integrity.

Examples used in this assignment:

- NOT NULL
- UNIQUE
- CHECK
- DEFAULT
- PRIMARY KEY

These constraints ensure the accuracy and reliability of data stored in the database.
