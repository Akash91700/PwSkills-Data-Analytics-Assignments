# Assignment Excel 04 - VLOOKUP, HLOOKUP & IF ELSE

## Overview
This assignment covers Excel lookup functions (`VLOOKUP`, `HLOOKUP`) and conditional logic using `IF`, `AND`, and `OR`.  
The task is to extract information from datasets, categorize salaries and performance, and generate output labels based on lookup and logical criteria.

## Skills Practiced
- VLOOKUP for vertical searches  
- HLOOKUP for horizontal searches  
- IF statements for conditional classifications  
- Combined IF + AND / IF + OR logic  
- Text concatenation formulas
