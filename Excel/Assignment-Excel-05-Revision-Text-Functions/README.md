# Assignment Excel 05 - Revision of TEXT Functions

## Overview
This assignment revises the use of Excel text functions to clean, extract, transform, and standardize textual data.  
The goal is to apply functions like TRIM, PROPER, LOWER, SUBSTITUTE, TEXT, MID, LEFT, RIGHT, SEARCH, and CONCATENATION to a dataset for practical text manipulation.

## Skills Practiced
- Text Cleanup & Standardization  
- Name and Email Parsing  
- Text Extraction  
- Domain Extraction  
- Pattern Standardization  
- Unique Identifier Creation
