# Assignment Excel 03 - Date and Time Functions

## Overview
This assignment covers Excel date & time functions like `TODAY()`, `NOW()`, and `DATEDIF()` along with their practical applications and real-world uses.  
It also includes formula examples to calculate durations, timestamps, service years, and dynamic messages.

## Topics Practiced
- TODAY() and NOW() functions  
- Date comparisons  
- Time calculations  
- Conditional formatting rules  
- DATEDIF for intervals  
- TEXT combined with date/time functions
