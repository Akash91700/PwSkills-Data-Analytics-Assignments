# Assignment Excel 02 - Text Functions

---

## Day 1: Basic Text Functions

**Objective:** Understand and apply fundamental text functions like LEFT, RIGHT, MID, and LEN.

**Tasks:**

1. Extract the first 5 characters from the string `"ExcelTipsAreGreat"` using the `LEFT` function.  
2. Extract the last 4 characters from `"DataAnalysis.xlsx"` using the `RIGHT` function.  
3. Extract the substring `"Tips"` from `"ExcelTipsAreGreat"` using the `MID` function.  
4. Count the total number of characters in `"Hello World!"` using the `LEN` function.  
5. Create a formula to extract the middle 6 characters from `"12345-67890-ABCDE"`.

---

## Day 2: Cleaning and Standardizing Text

**Objective:** Learn to clean and format text using `TRIM`, `UPPER`, `LOWER`, and `PROPER`.

**Tasks:**

1. Remove extra spaces from the text `"   Hello   Excel  "` using the `TRIM` function.  
2. Convert `"john doe"` to uppercase using the `UPPER` function.  
3. Convert `"MARY SMITH"` to lowercase using the `LOWER` function.  
4. Properly capitalize the text `"excEl FunCTions"` using the `PROPER` function.  
5. Combine `TRIM` and `PROPER` to clean and capitalize `"   messy text Example  "` in a single formula.

---

## Day 3: Combining Text

**Objective:** Understand how to combine text using `CONCAT`, `TEXTJOIN`, and `&` operator.

**Tasks:**

1. Use `CONCAT` to combine `"Hello"` and `"World"` with a space in between.  
2. Combine `"Apple"`, `"Banana"`, and `"Cherry"` into a single string separated by commas using `TEXTJOIN`.  
3. Use the `&` operator to create the string `"2025: Excel Functions"` by combining `"2025:"`, `" "` and `"Excel Functions"`.  
4. Create a comma-separated list from the range A1:A5 using `TEXTJOIN`.  
5. Combine first names in column A with last names in column B to create full names in column C.

---

## Day 4: Advanced Search and Replace

**Objective:** Explore `SEARCH`, `FIND`, `SUBSTITUTE`, and `REPLACE` functions.

**Tasks:**

1. Use `SEARCH` to find the position of `"l"` in `"Welcome to Excel"`.  
2. Use `FIND` to locate the position of `"Excel"` in `"Welcome to Excel"` (case-sensitive).  
3. Replace `"World"` with `"Excel"` in `"Hello World"` using `SUBSTITUTE`.  
4. Replace the year `"2020"` with `"2025"` in `"Budget_2020_Report.xlsx"` using `REPLACE`.  
5. Write a formula to find the position of `"-"` in `"ABC-123-XYZ"` and extract the first part of the string.
