# Assignment Excel 02 - Text Functions

## Overview
This assignment covers text manipulation functions in Excel including extracting parts of strings, cleaning and formatting text, combining text, and advanced search and replace operations.  
The solutions are provided in the accompanying Excel workbook.

## Skills Practiced
- LEFT, RIGHT, MID, LEN
- TRIM, UPPER, LOWER, PROPER
- CONCAT, TEXTJOIN, &
- SEARCH, FIND, SUBSTITUTE, REPLACE
- TEXT, VALUE, CLEAN, EXACT
